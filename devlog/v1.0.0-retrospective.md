v.1.0.0  开发者记。

这个项目是周六晚（4/11/2026）开工的，大概花了一天做到了现在这个样子。期间花了一些时间在日志，版本管理系统上，初步成品跑通后修了一下网页的显示排版，基本能正常渲染出 LaTeX（除了零散几个 /n 遗留）。想到这是我的项目开发处子作，于是仪式性地写这么一篇小记。

开发的动机是为了报比赛；区里要求报人工智能大赛赛道5的学生也要交一个赛道2的智能体作品用来筛选。虽然这个要求本身有点让人想敷衍过去（毕竟和我真正参加的赛道关系很小），但转念一想也正好就这个机会来入门一下项目开发。于是先构思、选题，然后在Chat界面里让 Opus 帮忙确定技术框架和开发流程（也就是 SPEC.md 和 PLAN.md），最后启动Claude Code 开始干活，整体比较顺利。

目前打算进一步做的事，包括：确认一下其适用科目的范围（跑一些非数理化的考试看下效果）；做国内考试适配（涉及到模型切换），不太确认美国模型是否擅长搜内网：做个更好的交互界面；增加搜考试信息的功能；等等

然后发现开始的指导文档写的有点死，为了后续更新还得重写一下各个md文件，又得搞一会儿了

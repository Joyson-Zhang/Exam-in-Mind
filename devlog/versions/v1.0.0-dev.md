## 版本总结

v1.0.0 是 Exam-in-Mind 的首个稳定版本，实现了从命令行输入考试名称到输出完整知识树静态站的全自动流程。核心功能包括：Brave Search 联网查考纲、Claude tool use 三级知识树构建、LeafContent 内容生成、MkDocs Material 站点渲染。通过 AP Calculus BC（378 知识点）和 SAT Math（145 知识点）两次端到端验证，证明系统具备跨科目泛化能力。

工程基础设施方面，建立了完整的文档体系（SPEC.md / PLAN.md / CLAUDE.md / devlog），8 阶段开发计划全部按序完成并通过验收。断点续跑机制在实际运行中经受了中断恢复的考验。

遗留问题：run.log 未实现（SPEC Section 11 要求）、KaTeX 依赖 CDN（SPEC 要求离线可用）、main.py 顶部注释过时、知识点数量对小科目偏多。这些将在 v1.0.1 中逐步解决。

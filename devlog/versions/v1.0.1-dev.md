## 2026-04-12

### `fix(renderer): MkDocs 站点 file:// 协议下链接无法跳转` · commit `e35f182`
**问题**: 双击 index.html 打开站点时，侧边栏链接跳转到文件系统目录索引而非页面内容。
**做法**: 在 mkdocs_renderer.py 中设置 `use_directory_urls: false`，让链接直接指向 .html 文件。
**效果**: ✅ 侧边栏链接可在 file:// 协议下正常跳转。

### `fix(content): formulas 字段被序列化为 JSON 数组字符串导致公式无法渲染` · commit `e35f182`
**问题**: Claude 返回的 formulas 中，62/145 个叶子节点的公式被整体序列化为一个字符串（如 `'["$x^2$", ...]'`），导致页面显示原始 LaTeX 代码。
**做法**: 在 content_builder.py 的 `_parse_leaf_content` 中增加 `_flatten_formula_list()`，用 json.loads + ast.literal_eval 后备解析展平数组字符串。
**效果**: ✅ 62 个异常公式全部修复，公式总数从 485 → 734（展平后）。
